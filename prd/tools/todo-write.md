# Todo Write Tool

## 基本信息

| 属性 | 值 |
|------|------|
| 工具名称 | `TodoWrite` |
| 源码位置 | `src/tools/TodoWriteTool/TodoWriteTool.ts` |
| 类型 | 任务清单工具 |

## 功能描述

结构化的 todo/任务列表管理。用于管理复杂多步骤任务。

## 使用场景

**应使用的场景**:
- 复杂多步骤任务 (3+ 步骤)
- 需要跟踪进度

**不应使用的场景**:
- 单一简单任务

## 任务状态

| 状态 | 说明 |
|------|------|
| `pending` | 待处理 |
| `in_progress` | 进行中 |
| `completed` | 已完成 |

## 提示词要点

```
- 任务需要同时提供:
  - 祈使语气的内容描述 (content)
  - 现在进行时的描述 (activeForm)
- 详细的使用/不使用示例及推理
```
