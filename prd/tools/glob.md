# Glob Tool

## 基本信息

| 属性 | 值 |
|------|------|
| 工具名称 | `Glob` |
| 源码位置 | `src/tools/GlobTool/GlobTool.ts` |
| 类型 | 文件搜索工具 |
| 只读 | 是 |

## 功能描述

快速文件模式匹配工具,适用于任何规模的代码库。支持 glob 模式匹配,返回按修改时间排序的文件路径。

## 输入参数

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `pattern` | string | 是 | glob 匹配模式 |
| `path` | string | 否 | 搜索目录 (默认当前工作目录) |

## 输出结构

```json
{
  "durationMs": 42,
  "numFiles": 15,
  "filenames": ["..."],
  "truncated": false
}
```

## 提示词要点

```
- 快速文件模式匹配工具,适用于任何代码库规模
- 支持 glob 模式如 "**/*.js" 或 "src/**/*.ts"
- 返回按修改时间排序的匹配文件路径
- 用于按名称模式查找文件
- 开放式搜索可能需要多轮 glob 和 grep 时,改用 Agent 工具
```

## 实现细节

- 默认限制 100 个结果
- 路径相对化处理
- 并发安全
