# Configuration Models

## Overview

Models for settings management and permission rules.

| Module | Model | Description |
|--------|-------|-------------|
| [Settings](settings/model-settings.md) | Settings | Multi-layered configuration for Claude Code behavior |
| [Permission](permission/model-permission-rule.md) | Permission Rule | A rule that controls tool execution permissions |
| [Permission](permission/model-hook.md) | Hook | A deterministic automation triggered by events |
