# Core Models

## Overview

Fundamental models powering the conversation engine, tool system, and task management.

| Module | Model | Description |
|--------|-------|-------------|
| [Conversation](conversation/model-session.md) | Session | A conversation session between the user and Claude |
| [Conversation](conversation/model-message.md) | Message | An individual message within a conversation |
| [Conversation](conversation/model-conversation-history.md) | Conversation History | Persistent record of a session's messages |
| [Tool System](tool-system/model-tool.md) | Tool | A capability that Claude can invoke |
| [Tool System](tool-system/model-tool-result.md) | Tool Result | The output of a tool invocation |
| [Task Management](task-management/model-task.md) | Task | A trackable work item within a session |
| [Prompt System](prompt-system/model-system-prompt.md) | System Prompt | Assembled instruction set with static/dynamic caching |
