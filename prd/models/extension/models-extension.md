# Extension Models

## Overview

Models for the extensibility layer — skills, plugins, and MCP integration.

| Module | Model | Description |
|--------|-------|-------------|
| [Skill](skill/model-skill.md) | Skill | A user-defined or bundled slash command |
| [Plugin](plugin/model-plugin.md) | Plugin | A third-party extension package |
| [MCP](mcp/model-mcp-server.md) | MCP Server | An external tool server connected via Model Context Protocol |
| [MCP](mcp/model-mcp-resource.md) | MCP Resource | A resource exposed by an MCP server |
