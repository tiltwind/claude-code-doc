# Task Status

## Overview

Lifecycle states for tasks managed within a Claude Code session.

## Values

| Value | Label | Description | Sort Order | Notes |
|-------|-------|-------------|------------|-------|
| pending | Pending | Task has been created but work has not started | 1 | Initial state |
| in_progress | In Progress | Task is actively being worked on | 2 | Shows spinner with activeForm text |
| completed | Completed | Task has been successfully finished | 3 | Terminal state |
| deleted | Deleted | Task has been permanently removed | 4 | Terminal state; task is no longer visible |
