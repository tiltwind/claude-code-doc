# Voice State

## Overview

States of the voice input system during push-to-talk interaction.

## Values

| Value | Label | Description | Sort Order | Notes |
|-------|-------|-------------|------------|-------|
| idle | Idle | Voice system is ready but not recording | 1 | Default state |
| recording | Recording | Audio is being captured from the microphone | 2 | Shows audio level indicator |
| processing | Processing | Captured audio is being transcribed to text | 3 | Shows processing spinner |
