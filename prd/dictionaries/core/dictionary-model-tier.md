# Model Tier

## Overview

Available Claude model tiers that users can select for their sessions.

## Values

| Value | Label | Description | Sort Order | Notes |
|-------|-------|-------------|------------|-------|
| opus | Opus 4.6 | Most capable model; best for complex reasoning and coding | 1 | Default for demanding tasks |
| sonnet | Sonnet 4.6 | Balanced capability and speed | 2 | Good for general tasks |
| haiku | Haiku 4.5 | Fastest model; best for quick, simple operations | 3 | Lower cost per token |
| opus-1m | Opus 4.6 (1M) | Opus with extended 1M token context window | 4 | For large codebases |
| sonnet-1m | Sonnet 4.6 (1M) | Sonnet with extended 1M token context window | 5 | For large codebases |
