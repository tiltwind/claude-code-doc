# Tool Category

## Overview

Categories that group built-in tools by their primary function.

## Values

| Value | Label | Description | Sort Order | Notes |
|-------|-------|-------------|------------|-------|
| file-io | File I/O | Tools for reading, writing, and editing files | 1 | FileRead, FileWrite, FileEdit, NotebookEdit |
| search | Search | Tools for finding files and searching content | 2 | Glob, Grep, ToolSearch |
| shell | Shell Execution | Tools for executing shell commands | 3 | Bash, PowerShell, Tungsten (virtual terminal) |
| web | Web Access | Tools for searching and fetching web content | 4 | WebSearch, WebFetch |
| agent | Agent & Collaboration | Tools for spawning agents and inter-agent communication | 5 | Agent, SendMessage, Team tools |
| task | Task Management | Tools for creating and managing tasks | 6 | TaskCreate, TaskUpdate, TaskGet, TaskList |
| planning | Planning | Tools for structured planning mode | 7 | EnterPlanMode, ExitPlanMode |
| mcp | MCP Integration | Tools for interacting with MCP servers | 8 | MCPTool, ListMcpResources, ReadMcpResource |
| scheduling | Scheduling | Tools for cron-based scheduled execution | 9 | CronCreate, CronDelete, CronList, RemoteTrigger |
| code-intelligence | Code Intelligence | Tools for language-aware code analysis | 10 | LSP |
| worktree | Git Worktree | Tools for isolated git worktree operations | 11 | EnterWorktree, ExitWorktree |
| config | Configuration | Tools for updating settings | 12 | ConfigTool |
| user-interaction | User Interaction | Tools for prompting the user | 13 | AskUserQuestion |
